# Web-Programming-workshop-2

**Workshop 2 - HTML**
รายวิชาการโปรแกรมบนเว็บ 2/2566
2 คะแนน

**วัตถุประสงค์**
1. นักศึกษาสามารถใช้งานภาษา HTML ในการพัฒนาเว็บไซต์ในส่วน front-end ได้อย่างถูกต้อง
2. นักศึกษาเข้าใจหลักการเขียนโปรแกรมด้วยภาษา HTML
3. นักศึกษาสามารถใช้ tag ทั้งแบบมี tag ปิดและไม่มีได้

**เนื้อหาที่เกี่ยวข้อง**
1. บทที่ 2-3
2. แบบฝึกหัดท้ายบทที่ 2-3

**คำชี้แจง**
1. ให้นักศึกษาเขียนเว็บเพจสำหรับ portfolio หรือแนะนำตัวเอง โดย
2. มีจำนวนเพจอย่างน้อย 3 เพจ<br>
   2.1. ในแต่ละเพจจะต้องมีการใส่รูปภาพ ข้อความ ลิงค์ และเมนู<br>
   2.2  ให้ใช้เฉพาะภาษา HTML เท่านั้น
   2.2. ทุกเพจจะต้องถูกพัฒนาตามมาตรฐานของ W3C ด้าน Web Design<br>
   2.3. ทุกเพจจะต้องมีการเขียนโปรแกรมไม่ต่ำกว่า 50 บรรทัด<br>
3. มี comment อธิบายการทำงานในทุกส่วนของโค้ด
4. ห้ามให้เว็บไซต์ของนักศึกษามีความเหมือนของคนอื่นๆ หรือบ่งบอกได้ว่าลอกกันมาหรือเอาโค้ดของคนอื่นมาปรับแก้ไขเป็นของตนเอง
5. นักศึกษาที่ตอบคำถามของอาจารย์ในขณะส่งงานไม่ได้ จะถือว่าไม่ได้เขียนโค้ดเอง และจะไม่ได้คะแนน
